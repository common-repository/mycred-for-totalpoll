== Description ==

[TotalPoll](https://1.envato.market/vKKJj) is a WordPress plugin that let you create and integrate polls easily, TotalPoll is made to be easy and powerful, Although it’s very simple to use, It provides big number of options and features to have full control over the polls. This plugin allows you to reward users with points for participating in polls.

= myCred for TotalPoll Features =
Here are the most notable features of myCred for TotalPoll.

* Award points for voting in polls.
* Badge support.
* Translation ready.

* [myCred 1.8+](https://wordpress.org/plugins/mycred/)

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for TotalPoll
3. Find myCred for TotalPoll and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
